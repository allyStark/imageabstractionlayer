Image Abstraction API


To Search:

To search for images: <thisaddress>/api/<your search term>

To add an offset to the results (optional): <thisaddress>/api/<your search term>?offset=<number to offset by>

Example: www.thisapp.heroku.com/api/northern lights/?offset=10


To Check Previous Queries:

<thisaddress>/queries

Example: www.thisapp.heroku.com/queries


Enjoy!
